🤖 Automação de Cadastro de Produtos com Python
🧠 Contexto do Projeto

Este projeto foi desenvolvido para praticar automação de tarefas repetitivas utilizando Python.
A aplicação simula o cadastro automático de produtos em um sistema web, lendo informações de um arquivo CSV e preenchendo formulários de forma automática no navegador.

O sistema utilizado é uma plataforma educacional da Hashtag Treinamentos.

🎯 Objetivo

Automatizar o preenchimento de formulários web

Integrar automação com leitura de dados

Reduzir tarefas manuais repetitivas

🚀 Funcionalidades

Abertura automática do navegador

Login automatizado no sistema

Leitura de dados via CSV

Preenchimento automático de formulários

Tratamento de campos vazios

Cadastro sequencial de múltiplos produtos

🛠️ Tecnologias Utilizadas

Python

PyAutoGUI

Time
