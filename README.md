# 🤖 Automação de Cadastro de Produtos com Python

## 🧠 Sobre o Projeto
Este projeto foi desenvolvido com o objetivo de praticar **automação de tarefas repetitivas** utilizando Python.  
A aplicação simula o **cadastro automático de produtos em um sistema web**, lendo os dados a partir de um arquivo CSV e preenchendo os campos de forma automática no navegador.

O sistema utilizado para teste é uma plataforma educacional da :Hashtag Treinamentos.

---

## 🚀 Funcionalidades
- Abertura automática do navegador (Google Chrome)
- Login automático no sistema web
- Leitura de dados a partir de um arquivo CSV
- Preenchimento automático de formulários
- Tratamento de campos vazios (ex: observações)
- Cadastro sequencial de múltiplos produtos

---

## 🛠️ Tecnologias Utilizadas
- **Python**
- **PyAutoGUI** – automação de mouse e teclado
- **Pandas** – leitura e manipulação de dados
- **Time** – controle de tempo entre ações
